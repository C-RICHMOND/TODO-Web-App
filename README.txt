Installation:
1. Download the project from github.
2. Run the database script through terminal or command line to set up the database.
3. Add your mysql username and password to DbConfig.php
4. If using WAMPServer, put project files within the 'www' folder. Then navigate to 'localhost/ToDo_Application'
   in your web browser.
5. Otherwise, use IDE of choice and run the application (set index.php as start page).
